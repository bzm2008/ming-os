# Ming OS 26.4.0 正式版官网与 GitHub Release 文案

Ming OS 26.4.0 正式版提供更稳定的软件安装、无线联网和前台响应。公开显示版本为 26.4.0，事务版本为 26.4.0.1。正式 ISO、官网入口与事务 OTA 在签名发布材料和升级回归完成前保持关闭。

升级流程会显示真实事务状态：签名校验、暂存、待重启、健康检查、完成或自动回滚。系统不会把“已下载”误报为“已更新”，也不会引导用户绕过签名、空间、/home 保留或 recovery ISO 独立备份介质门禁。

## 发布资产

- 26.4.0.1 签名 manifest、content index、payload 及 detached signatures（事务 OTA 开放前必须补齐）；
- 26.3.2 官方签名 bootstrap、SHA256、detached signature 和公钥指纹；
- 最终 ISO 与 SHA256：`ming-os-26.4.0-home-amd64-formal.iso`，`<FINAL_ISO_SHA256_AFTER_BUILD>`；
- GitHub 标签为 `v26.4.0.1`；发布前官网和事务 OTA discovery 均保持安全关闭。

发布前请确认 `from_versions: ["26.3.2", "26.3.3", "26.4.0", "26.4.0.1-development"]` 与 `version: "26.4.0.1"` 同时存在于签名 manifest 中。
