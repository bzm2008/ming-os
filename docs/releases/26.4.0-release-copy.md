# Ming OS 26.4.0 官网与 GitHub Release 文案

Ming OS 26.4.0 跳过 26.3.3，提供更稳定的安装、无线联网、软件安装和事务型 OTA。26.3.2 用户只需安装一次官方签名 bootstrap，之后在 Ming 设置中点击“检查更新”，即可发现并自动完成 26.4.0 升级。

升级流程会显示真实事务状态：签名校验、暂存、待重启、健康检查、完成或自动回滚。系统不会把“已下载”误报为“已更新”，也不会引导用户绕过签名、空间、/home 保留或 recovery ISO 独立备份介质门禁。

## 发布资产

- 26.4.0 签名 manifest、content index、payload 及 detached signatures；
- 26.3.2 官方签名 bootstrap、SHA256、detached signature 和公钥指纹；
- 最终 ISO/VHDX 与 SHA256（完成构建和 BIOS/UEFI 验证后填写）；
- GitHub Release 与官网下载地址（完成发布后填写）。

发布前请确认 `from_versions: ["26.3.2"]` 与 `version: "26.4.0"` 同时存在于签名 manifest 中。
