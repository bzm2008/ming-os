# Ming OS 26.4.0 官网与 GitHub Release 文案

Ming OS 26.4.0 跳过 26.3.3，提供更稳定的安装、无线联网和软件安装。官网 ISO 下载已开放；事务型 OTA 在签名发布材料完成前保持安全关闭。

升级流程会显示真实事务状态：签名校验、暂存、待重启、健康检查、完成或自动回滚。系统不会把“已下载”误报为“已更新”，也不会引导用户绕过签名、空间、/home 保留或 recovery ISO 独立备份介质门禁。

## 发布资产

- 26.4.0 签名 manifest、content index、payload 及 detached signatures（事务 OTA 开放前必须补齐）；
- 26.3.2 官方签名 bootstrap、SHA256、detached signature 和公钥指纹；
- 最终 ISO 与 SHA256：`ming-os-26.4.0-home-amd64.iso`，`ab76033a215b13debb89eec8bb43a72a43d095ddd85ca1e563067680b4d5f11d`；
- GitHub Release 与官网下载地址已发布；事务 OTA discovery 当前为 `delivery:none`。

发布前请确认 `from_versions: ["26.3.2"]` 与 `version: "26.4.0"` 同时存在于签名 manifest 中。
